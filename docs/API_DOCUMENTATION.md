# BYOT — REST API

OpenAPI 3.1 spec is auto-generated by FastAPI at:

```
GET /openapi.json
GET /docs              # Swagger UI
GET /redoc             # ReDoc
```

This document is a human-friendly summary.

## 1. Conventions

* Base URL — `https://api.byot.earth/api/v1`
* All requests/responses are JSON unless noted.
* Auth — `Authorization: Bearer <access_token>`
* Pagination — `?page=1&page_size=50` → `{items, page, page_size, total}`.
* Sorting — `?sort=-created_at`
* Filtering — RHS-colon (`?status:eq=active`, `?registered_at:gte=2025-01-01`).
* Idempotency — mutating endpoints accept `Idempotency-Key` header.
* Rate limit — `1000 req / 15 min / user` (defaults), per-endpoint overrides.

### 1.1 Error model

```json
{
  "error": {
    "code": "tree_not_found",
    "message": "Tree with id 7c… does not exist",
    "details": { "tree_id": "7c…" },
    "trace_id": "01HXXX…"
  }
}
```

## 2. Authentication

| Method | Path | Description |
|---|---|---|
| POST | `/auth/register` | Create user (email + password) |
| POST | `/auth/login` | Email/password → access + refresh tokens |
| POST | `/auth/refresh` | Refresh access token |
| POST | `/auth/logout` | Revoke refresh token |
| POST | `/auth/otp/request` | Send OTP via email/SMS |
| POST | `/auth/otp/verify` | Verify OTP → tokens |
| GET  | `/auth/google/login` | Begin OAuth |
| GET  | `/auth/google/callback` | Complete OAuth |
| GET  | `/auth/me` | Current user profile |
| PATCH| `/auth/me` | Update profile |

```http
POST /api/v1/auth/login
{ "email": "ada@example.com", "password": "s3cret!" }

200
{ "access_token": "...", "refresh_token": "...", "token_type": "Bearer", "expires_in": 900 }
```

## 11. Webhooks (outbound)

* `tree.registered`, `analysis.completed`, `satellite.updated`,
  `carbon.recalculated`, `alert.created`
* HMAC-SHA256 signature in `X-BYOT-Signature` header.

## 12. Versioning

* URI versioning (`/api/v1`).
* Deprecations announced via `Sunset` header ≥ 6 months in advance.
